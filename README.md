# StudyConnect Contribution

This repository contains my individual contribution for the CSP1123 Mini IT Project.

## My Role

I was Member 1, responsible mainly for the GUI, user flow, profile page, search page, and integration with the matching and database system.

## My Contribution

I worked on:

- Login interface
- Profile page
- Profile view and update feature
- Search interface
- Matching result display
- Database integration using SQLite
- Input validation
- Top 3 match display
- Best match highlight
- No-match message
- Clear results button
- User-friendly navigation

## Files

- `main.py`  
  Handles the GUI, page navigation, profile page, search page, and result display.

- `database.py`  
  Handles SQLite database functions such as saving, updating, and retrieving user profile data.

- `matching.py`  
  Handles connection to the matching logic and match result calculation.

## Final Group Project

The final deployed group project was adapted into a web application by the team.  
My contribution focused on the prototype, user flow, profile update logic, search flow, validation, and matching result display.

## Purpose

This repository is used to show my individual contribution and development work for the project.
