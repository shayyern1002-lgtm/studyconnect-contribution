This folder contains the original prototype files developed for the CSP1123 Mini IT Project.

My contribution focused on:

- GUI and user flow
- Profile page
- Profile update functionality
- Search interface
- Matching result display
- Database integration
- User input validation

Files:

- `main.py` – Main GUI and application flow
- `database.py` – Database operations and data storage
- `matching.py` – Matching logic integration and result handling

These files represent my original development work before the project was adapted into a web application.
